## 基础模型

前端基础模型，主要是基于 IBizSys 预置模型结构化组织应用模型。

基础模型主要提供前端应用发布所需的预置模型结构，生产前端对应的业务模型内容。

基础模型结构如下：

![基础模型](/imgs/base-model/base-model.png)

基础模型层级主要分为四层：
- App (应用) 
- View (视图)
- Control (部件)
- Item (部件成员或者编辑器)
